السادة أعضاء لجنة التحكيم الموقرين،

لتشغيل منصة "مَلاذ" وربطها بقواعد البيانات المرفقة:

لمستخدمي نظام Windows:
1. يرجى التأكد من وجود لغة Python مثبتة على الجهاز.
2. اضغطوا مرتين على الملف التنفيذي (Run_Malath.bat).
3. سيقوم الملف برمجياً بتحميل المكتبات اللازمة وفتح المنصة في متصفحكم.

لمستخدمي نظام Mac:
يمكنكم تشغيل المنصة بفتح موجه الأوامر (Terminal) داخل هذا المجلد وكتابة:
pip3 install -r requirements.txt
ثم:
python3 -m streamlit run app.py

ملاحظة: قاعدة البيانات الإحصائية (CSV) متوفرة داخل مجلد Data.
